# Xamanとは？

秘密鍵をXamanは高い安全性を誇る、XRPレジャーNo.1の自己管理型ウォレットです。

ユーザーの秘密鍵はデバイスに安全に保管しながら、XRPレジャーの機能すべてを楽しめる特別なアプリです。

Xamanの紹介ビデオを制作しましたので、ぜひご覧ください。
